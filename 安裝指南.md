# 安裝指南

這個空專案是基於 DCat Admin 2 修改一部份出來的，都沒有改到原始開發者的設計都可以自由的升級


## 操作

1. 進行資料庫遷移
```bash
php artisan migrate
```
2. 進行後台基本資料安裝
```bash
php artisan admin:install
```
3. seeder 補充資料 進行最後的資料調整
```bash
php artisan db:seed --class=Database\Seeders\Admin\InitConfigSeeder
```
